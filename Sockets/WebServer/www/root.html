You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html

  • /json -- returns a json of the /random request

  • /random -- returns index.html

  • /multiply?num=X&num2=Y -- returns the product of integers X and Y
    Example: /multiply?num1=21&num2=2

  • /github?query=users/USER/repos -- returns the USER's list of GitHub repos
    Example: /github?query=users/amehlhase316/repos

  • /compatible?name1=X&name2=Y -- return the compatibility between person X and person Y
    Example: /compatible?name1=Romeo&name2=Juliet

  • /chat?name=X&msg=Y -- return a chat log and appends person X's message Y
    Example: /chat?name=Test&msg=This is a test
